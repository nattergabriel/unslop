# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this is

unslop is an open source Claude skill that keeps AI artifacts (slop) out of everything Claude produces: prose, code, frontend design, docs, and naming. Existing anti-slop skills only cover prose; covering all surfaces in one skill is the point.

This is a skill project, not an application. There is no build, lint, or test toolchain. Evals come in phase 6 of the plan.

PLAN.md is the source of truth for scope, design decisions, and phases. Read it and check which phase is current before starting work.
